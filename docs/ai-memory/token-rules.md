# Token Rules

- Never scan the whole repo unless necessary.
- Read `docs/ai-memory/session-handoff.md` first.
- Read `AGENTS.md` second.
- Read only files relevant to the current error or task.
- Prefer grep/ripgrep/search before opening large files.
- Summarize after each major step.
- Before starting a new task, write/update `docs/ai-memory/session-handoff.md`.
- Before editing, explain targeted files.
- Do not paste large file contents into chat unless asked.
- Keep memory docs short; store stable facts, not temporary noise.
- Use `/compact-context` before long sessions grow too large.
